## Unlocks Per Rank

### Rank A (Slave) — Starting Rank
- Access: Pit of Souls (starter mine)
- Commands: `/sell`, `/quests`, `/ranks`, `/mines`, `/leaderboard`, `/kit starter`
- GUI access: Main Menu, Help

### Rank B (Serf)
- Access: Tomb of Aten
- Sell streak now tracks (streak system active from first sell)

### Rank C (Laborer)
- Access: Tomb of Thoth (minor improvement, same mine group as Serf)

### Rank D (Craftsman)
- Access: Tomb of Thoth fully unlocked (if gated by D not C)
- Chat prefix color changes to white

### Rank E (Merchant) ★ First Milestone
- Access: Tomb of Sobek
- Unlocks: `/ah` (Auction House)
- **Kit:** rankE — Iron gear set
- Mine quality jump: Gold ore appears for first time

### Rank G (Soldier) ★ First Wall
- Access: Tomb of Hapi
- Unlocks: Dynasty (Gang) creation (`/gang create`)
- Chat prefix color changes to yellow
- **This is the first major cost wall. Players who push through are invested.**

### Rank J (Architect) ★ Second Milestone
- Access: Tomb of Sekhmet
- Unlocks: Higher Khopesh enchant tiers (if gated)
- **Kit:** rankJ — Diamond gear set
- Chat prefix color changes to gold

### Rank N (High Priest) ★ Mid-Game
- `/prestige` command becomes visible in menu (can't use yet — need Rank Z)
- Access: Tomb of Ptah (Diamond + Emerald first appears)

### Rank O (Sage) ★ Third Milestone
- Access: Tomb of Seth (Diamond + Emerald dominant)
- **Kit:** rankO — Diamond + Mending gear
- Chat prefix color changes to green

### Rank T (Horus-Born) ★ Late Game Entry
- Access: Tomb of Horus (Netherite scrap appears)
- **Kit:** rankT — Full Diamond + Mending + Unbreaking gear
- Chat prefix color changes to light purple

### Rank U (Champion)
- Access: Tomb of Ra (significant Netherite scrap)
- Chat prefix: light_purple → red transition

### Rank W (Immortal)
- Access: Tomb of Osiris (Ancient Debris appears)

### Rank Z (Pharaoh) ★ Max Rank
- Access: Tomb of Anubis (pure Ancient Debris — highest value mine)
- Unlocks: `/prestige ascend` (Ascension available)
- **Kit:** rankZ — Full Diamond + Mending + Unbreaking + high enchants
- Broadcast to all players on server
- Exclusive cosmetic: `[Pharaoh]` chat tag unlocked
- Chat prefix: bold gold — visually distinct from all other ranks

## Permission Nodes

Each rank grants a permission node: `prison.mine.[letter]` (lowercase).

| Rank | Permission |
|---|---|
| A | `prison.mine.a` |
| B | `prison.mine.b` |
| ... | ... |
| Z | `prison.mine.z` |

These permissions are cumulative — Rank J has all permissions from A through J.
Mine access is checked by `prison.mine.[letter]` in the mine's config.
